# 德州扑克游戏项目

## 项目概述
这是一个基于Python实现的德州扑克游戏，支持人机对战和AI策略。

## 功能特点
- 支持多人游戏和AI对手
- 完整的德州扑克规则实现（翻牌前、翻牌、转牌、河牌）
- AI智能决策系统
- 游戏状态实时显示

## 运行方法
```bash
python main.py
```

## 依赖项
- Python 3.x
- random模块

## 游戏规则
1. 每个玩家发两张底牌
2. 依次进行翻牌前、翻牌、转牌、河牌阶段
3. 每个阶段后都有下注环节
4. 最后比较牌型决定胜负

## AI策略
AI会根据手牌强度和底池赔率做出决策，包括弃牌、跟注和加注。
